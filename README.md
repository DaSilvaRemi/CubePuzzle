# Game Instructions

Contrôles

- Z / Up Arrow : Avancer
- S / Down Arrow : Reculer
- Q / Left Arrow : Tourner vers la gauche
- D / Right Arrow : Tourner vers la droite
- R  : Redémarrer le niveau
- C : Changer de caméra
- P : Pause
- Space : Sauter
- Ctrl / Left Click : Tirer (disponible à partir du LVL 3)


## Condition de victoire
Terminer le dernier niveau en touchant la clé. Un niveau se gagne en atteignant une plaque de pression ou en trouvant le coffre.

## Condition de défaite 
Être tué par un ennemi, se noyer, compte à rebours écoulé.


## Capacités du player :
Selon le niveau, le player peut marcher, sauter et tirer afin de tuer des ennemis, franchir des obstacles et gagner du temps et du score en collectant des bonus.


# Remarques particulières, importantes :
Nous n’avons pas suivi les modèles proposés qui nous avaient été envoyés par mail. Nous avons donc conçu notre propre architecture à l’aide des cours d’Unity afin d’assurer la couverture de notre compréhension à 100 %.


# Limitations connues : 
- Quelques pertes de performances sur certains appareils de faible puissance
- “L’intelligence artificielle” des ennemis est très limitée étant donné que les ennemis se contentent de suivre la position du joueur. Un ennemi peut se trouver coincé derrière un mur
- Les chargements des scènes peuvent être lents notamment le dernier niveau


# Bugs connus : 
-Le joueur peut traverser les murs s’il est insistant dû à la surcharge des calculs du moteur physique


# Voies d’amélioration :
- Proposer des niveaux bonus
- Mettre des FX de light
